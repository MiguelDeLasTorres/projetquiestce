Générateur JSON pour Qui est-ce Groupe AF Beldjilali Maxime et Young Brun Luis-Miguel

/////////////////////////////////////////////////////////////////////////////// Installation:

Dépendance nécessaire : VM java SE-17 ou plus récent Le projet utilise des librairies javafx et Jackson incluses dans le .jar

/////////////////////////////////////////////////////////////////////////////// /////////////////////////////////////////////////////////////////////////////// Informations sur le générateur:

Prend en entrée un dossier contenant un certain nombre de d'image.

Permet de créer des attributs et de leurs associer des valeurs.

Permet d'associer un ensemble de couple Attribut X Valeur à un nom et une image, créant ainsi un Personnage. Le but étant d'obtenir ensuite un ensemble de personnage.

Convertit cet ensemble de Personnage en deux fichier Attribut.json et Personnage.json exploitable par le Qui est-ce.

/////////////////////////////////////////////////////////////////////////////// /////////////////////////////////////////////////////////////////////////////// Fonctionnement :

Pour lancer le générateur, lancer generateur.jar avec une VM java .

Pour charger un dossier d'image, appuyer sur le bouton "sélectionner un dossier".

Pour le nom, écrivez le dans le champs de texte "nom" puis appuyer sur la touche "Entrée".

Pour ajouter un attribut, écrivez le dans le champs de texte adéquat (indiquez par "attribut") puis appuyez sur le bouton "V". De même pour une valeur.

Appuyer sur le bouton "Valider".

Naviguer entre les différents personnages avec les flèches à droite et à gauche de l'image.

Une fois tous les attributs de tous les personnages remplit appuyer sur "sauvegarder". La sauvegarde est impossible si deux personnages ont exactement un même ensemble de Attribut X Valeur.
